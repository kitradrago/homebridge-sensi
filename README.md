# homebridge-sensi

Homebridge plugin for Sensi 2 thermostat.

## Configuration

- `name`: accessory name
- `refreshToken`: Sensi API refresh token (you must extract manually)
- `deviceId`: your thermostat device ID

## Features

- Read current & target temperature
- Set target temperature
- Read & set heat/cool/off mode

## Notes

This plugin uses the Sensi cloud API; local LAN control is not available because Sensi devices do not expose a local API.

## Disclaimer

This is an unofficial integration, use at your own risk.
